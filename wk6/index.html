Project Questions

Why is textContent safer than innerHTML?

The textContent property is safer because it adds content as plain text. The innerHTML property can interpret HTML code, which could allow unsafe scripts or unwanted HTML elements to be added to the page.

How are HTML elements connected to JavaScript variables using DOM selection?

HTML elements are connected to JavaScript variables by using DOM selection methods such as document.querySelector(). The selector locates an element by its ID, class, or tag and stores it in a JavaScript variable.

What would happen to the text box with id="text" if I added this line: text.textContent = "Hello!"?

The content inside the element with the ID of text would be replaced with the words Hello!
